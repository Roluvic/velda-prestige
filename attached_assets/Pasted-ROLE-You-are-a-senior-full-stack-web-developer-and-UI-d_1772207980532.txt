AI QUOTE TOOL (CORE FEATURE)
Build a guided flow with 2 modes:
A) Wizard form (structured)
B) AI chat assistant (conversational)
Both must converge into a FINAL structured “Recommended Configuration” object plus a “Quote Request Summary”.

Wizard fields:
- Application type (Lighting suspension, Safety/fall protection, HVAC/ducts, Signage, Architectural cables, Industrial, Other)
- Indoor/Outdoor, dry/wet environment, corrosion requirement (galvanised vs stainless)
- Load (kg) and number of suspension points
- Cable length (m)
- Angle of suspension (0/15/30/45/60) if applicable
- Need for adjustability (yes/no)
- Required end fittings (choose or “not sure”)
- Certifications/constraints (optional)
- Quantity
- Attach drawing/spec (optional file upload placeholder)

AI assistant guardrails:
- Always ask clarifying questions until enough info is collected.
- Must follow known constraints (examples):
  - QS products: indoor dry environment only; never for lifting; never exceed 60°; working load changes under angle.
  - Use safety factors and show conservative recommendations.
- Must output:
  1) Recommended cable type and diameter (with reasoning)
  2) Suggested finishes/components
  3) Warnings and assumptions
  4) A clean bill of materials style list

After completion:
